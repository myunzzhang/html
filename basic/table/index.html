표만들기

html에서 표를 구현할때 사용하는 태그

  • tr(table row):행
  • td(table data):(행의 내용)열
  • th(table heading):행의 열의 제목(굵은체, 가운데정렬)
  • 속성 colspan="2" : 2칸의 열병합
  • 속성 rowspan="3" : 3칸의 행병합
